# fuzzley.github.io

This repository hosts the production build of the personal/portfolio website for Andrey Gavel.

## ⚠️ Sync Notice

Do **not** edit the files in this repository directly. 

This repository is automatically updated and synced by a GitHub Actions workflow from the [fuzzley](https://github.com/fuzzley/fuzzley) repository whenever changes are merged into its `main` branch.

All development, bug fixes, and styling updates should be made in the [fuzzley](https://github.com/fuzzley/fuzzley) repository.
